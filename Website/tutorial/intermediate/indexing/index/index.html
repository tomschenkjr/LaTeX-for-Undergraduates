Topics covered in this slide:
  • Introducting the makeidx package.
  • Creating index points and the index.
  • Parent-child index entries.
Along with the table of contents and bibliographies, creating indexes is simple. Normally indexing would require the author to keep track of certain words and pages they are written. To create an index in LATEX we need to use the makeidx package. Also, we need to place \makeindex in the preamble. To create actual points we want in the index, we use \index{key} immediately after the word we're referencing, where key is the word or phrase we want to use in the index. We can create parent-child relationships using \index{key!sub-key}.
