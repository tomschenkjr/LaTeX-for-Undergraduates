A.2 R/GNU S Statistical Suite
Target Majors: Actuarial/Statistics, Economics, Mathematics, Engineering

GNU R is a free statistical suite modeled after the commercial S-plus statistical suite (developed by the same person). In this respect, GNU R is sometimes referred to as GNU S. The Sweave library allows users to write R code in a LATEX document. When the document is processed, the LATEX code will be interpreted as text, while the GNU R code will be recognized and outputed as well. That is, as the document is generated, your calculations are also being generated an inserted into your document (including graphs and figures)! Normally, graphs/figures/etc. are generated first and saved as .eps files. Those objects are then included using the \includegraphics command. Using Sweave, in conjunction with R and LATEX, it is a one-step process where graphs are generated and included as it is compiled. Tables can be generated under similar circumstances. Nevertheless, GNU R is a sophisticated and technical suite, so this is namely targeted towards those using complicated statistical analysis. Below is a link to the needed programs and websites explaining GNU R and Sweave.

GNU R

As previously mentioned, GNU R, or simply R, is an open-source implementation of S-plus that is compatible with Windows, Linux, and MacOS. Sometimes it is referred to as GNU S after it's commercial counterpart. However, it is a very technical and sophisticated suite. Instead of using menu's (e.g. a Graphical User Interface (GUI)), it is command line driven where the operator must know typed commands (screenshot). Nevertheless, if you are apt to using sophisticated statistical analysis (e.g. OLS regression, Probit/Tobit, panel data, etc.), I strongly suggest that you use this program. You can get a better overview of the program at The R Project for Statistical Computing. You may also download the program from their downloads section. They have a handful of manuals available, my particular favorite is An Introduction to R.

Sweave

Along with R, you will need the Sweave library. This is the actual program that will parse the code for LATEX and R commands. Similar to LATEX, R uses packages to load extra features. Thus, Sweave is not an independent program, but a package loaded into R. A variety of documents is located at Sweave's website which are helpful. Particularly helpful is Sweave and Beyond, a conceptual overview, and the Sweave manual. Of course, it assumes you are familar with GNU R.

A Few Comments

Sweave requires that the author type her document into R, generally, this may not be desireable for the author. Nevertheless, (1) knowing R is very useful and (2) this is particularly handy when working with a lot of graphs or tables. I'd also like to thank Paolo Bosetti for his blog entry on Sweave.
