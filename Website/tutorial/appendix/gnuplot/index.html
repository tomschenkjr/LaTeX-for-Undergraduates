What is gnuplot?

Gnuplot is a data and function plotting program that can be used to create very sophisticated graphs. Normally, gnuplot is used in conjunction with other programs, such as GNU R, GNU Octave, or gretl to name a few. Similar to many powerful computing programs, gnuplot is a command-line driven interface where the graph's output depends on the arguments in the command. You may either plot mathematical functions (similar to Mathematica) or create graphs using a data set. Here are some examples of graphs.

Installation

Gnuplot is compatible with Windows, MacOS, Unix, Linux, Solaris, and other operating systems. You can obtain a copy from their downloads page. Normally, programs are "autoinstalled," where folders and icons are automatically placed in the computer and menus. Here, you are required to unzip the program and place it in the appropriate locations. For Windows, this is typically in C:\Program Files or for MacOS, in the Application folder. Please see their webpage for more help on installation.

Outputting to LATEX

David Klotz and the gnuplot development team have put together an excellent tutorial (PDF) on how to output a graph into LATEX. Their tutorial requires some working knowledge of gnuplot, but I've provided some links to gnuplot tutorials in the supplemental documents area. If you are wanting to plot mathematical functions, I suggest looking at Mathematica. However, if you do not have Mathematica or are interested in plotting data, I strongly recommending learning gnuplot.
